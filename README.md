PERT for Cost Risk Analysis — Interactive Learning Tool
MGMT 410: Project Management
Penn State Erie, The Behrend College · Sam & Irene Black School of Business

This interactive tool walks you through the four-step PERT (Program Evaluation and Review Technique) for project cost risk analysis:

Three-Point Estimates — Enter optimistic, most likely, and pessimistic costs for each task
PERT Formulas — See how the mean and variance are calculated
Sum of Random Variables — Combine individual task distributions into a total project cost distribution
Probabilistic Estimates — Answer budget risk questions using NORMDIST and NORMINV

How to Use
Open the tool: pert-cost-risk-analysis.html
The tool loads with the DTP (Design–Test–Production) example from the course LBD exercise. You can modify the default values, add or remove tasks, and explore how changes in estimates affect the project cost risk profile.

Reference
PMI. A Guide to the Project Management Body of Knowledge (PMBOK Guide).
